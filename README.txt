============================================================
  IMG AI Studio - 로컬 AI 이미지 스타일 변환
  RTX 5060 Ti 16GB 최적화
============================================================

[ 시스템 요구사항 ]
- CPU: AMD Ryzen 7 7800X3D (또는 동급)
- RAM: 16GB (가상메모리 32GB+ 권장)
- GPU: NVIDIA RTX 5060 Ti 16GB
- 저장공간: 50GB+ 여유 공간 (모델 저장용)
- OS: Windows 10/11 64bit

[ 사전 설치 필요 ]
1. Python 3.10 이상
   - https://www.python.org/downloads/
   - 설치 시 "Add Python to PATH" 반드시 체크!

2. NVIDIA 드라이버 (최신 버전)
   - https://www.nvidia.com/drivers
   - RTX 50시리즈는 최신 Game Ready 드라이버 필요

3. XAMPP (Apache + PHP)
   - 이미 설치되어 있음

[ 설치 방법 ]
1. setup.bat 더블클릭하여 실행
2. 설치 완료까지 대기 (10-20분 소요)
3. 첫 실행 시 AI 모델 다운로드 (10-15GB)

[ 사용 방법 ]
1. XAMPP Control Panel에서 Apache 시작
2. 브라우저에서 http://localhost/img_ai 접속
3. 카메라 촬영 또는 이미지 업로드
4. 스타일 선택 (애니메이션, 카툰, 지브리, 코믹)
5. "AI 변환 시작" 클릭
6. 30초~2분 대기 후 결과 확인

[ 스타일 설명 ]
- 애니메이션: 일본 애니메이션 스타일
- 카툰: 디즈니/픽사 3D 카툰 스타일
- 지브리: 스튜디오 지브리 스타일
- 코믹: 미국 만화책 스타일

[ 문제 해결 ]
1. "GPU를 찾을 수 없습니다"
   → NVIDIA 드라이버 최신 버전 설치
   → 그래픽카드가 제대로 장착되었는지 확인

2. "메모리 부족"
   → 가상메모리 32GB 이상 설정
   → 다른 프로그램 종료

3. "모델 로딩 실패"
   → 인터넷 연결 확인
   → 디스크 공간 확인 (50GB+ 필요)

4. PHP 오류 발생
   → php.ini에서 max_execution_time = 300 설정
   → memory_limit = 512M 설정

[ 파일 구조 ]
img_ai/
├── index.html      # 메인 웹 인터페이스
├── css/            # 스타일시트
├── js/             # JavaScript
├── php/            # PHP 백엔드
├── python/         # AI 처리 스크립트
├── uploads/        # 업로드된 원본 이미지
├── outputs/        # 생성된 이미지
├── models/         # AI 모델 (선택)
└── logs/           # 로그 파일

[ 성능 최적화 팁 ]
1. 입력 이미지는 1024x1024 이하 권장
2. 변환 강도 0.6-0.8 권장 (너무 높으면 원본 손실)
3. 생성 중 다른 GPU 작업 피하기
4. VRAM 사용량 모니터링 (작업 관리자)
